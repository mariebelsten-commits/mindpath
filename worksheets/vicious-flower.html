How to use this worksheet

Start with the centre box: write down the core statement or feeling that keeps coming back to you. Then add petals for each thought or behaviour that feeds into it and keeps it alive. Add as many petals as you need — and remove any that don't apply. There is no right or wrong number.

Map Your Vicious Flower

Fill in the centre first, then add a petal for each thought or behaviour that feeds back into the core feeling. Use the + button to add petals and × to remove any that don't apply.
